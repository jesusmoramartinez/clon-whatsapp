Maquetación UI estilo WhatsApp — Entrega

Descripción breve
Este proyecto es una maqueta de la interfaz de WhatsApp creada con HTML y CSS. La maqueta reproduce la estructura visual principal: un sidebar con lista de chats y un panel de conversación central. No se incluyó lógica en JavaScript; el foco está en la maquetación visual y la adaptación responsive básica.

Qué se reprodujo
- Sidebar con avatar, búsqueda y lista de chats.
- Panel principal con cabecera de conversación, mensajes y área de envío (estática).
- Iconos e imágenes usadas desde la carpeta de `assets`.

Tecnologías usadas
- HTML
- CSS (flexbox/grid para layout)

Estructura del proyecto
- `index.html`  — página principal con la maqueta.
- `style.css`  — estilos del proyecto.
- `assets/`  — carpeta con imágenes e íconos usados en la maqueta.

Cómo visualizarlo
Abrir `index.html` en cualquier navegador moderno (doble clic o arrastrar al navegador).

Decisiones de diseño / Consideraciones
- Colores: se seleccionaron tonos aproximados a la paleta de WhatsApp (verdes suaves y grises) para mantener contraste y legibilidad.
- Tipografía: se usó la tipografía por defecto del sistema para simplicidad; se puede añadir una fuente web (p. ej. Google Fonts) si se desea más fidelidad.
- Layout: se utilizó `flexbox` para el sidebar y el panel de conversación por su facilidad para distribuir columnas y filas, y `grid` donde se requirió un control más preciso de la rejilla de mensajes.
- Organización de estilos: estilos principales en `style.css`, con clases semánticas para componentes (sidebar, chat-list, chat-panel, message, etc.) para mantener separación y facilidad de mantenimiento.
